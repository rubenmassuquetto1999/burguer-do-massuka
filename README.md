# 🍔 Burguer do Massuka

Projeto desenvolvido para colocar em prática a manipulação de arrays em JavaScript, criando uma interface dinâmica para uma hamburgueria com design inspirado no estilo icônico do McDonald's.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica.
* **CSS3**: Design moderno, responsividade (Grid e Flexbox) e estilização temática.
* **JavaScript (ES6+)**: Lógica de programação e manipulação do DOM.
* **Google Fonts**: Tipografia personalizada para melhor experiência do usuário.

## 🛠️ Funcionalidades

O projeto foca no uso de métodos fundamentais de array para gerenciar o cardápio:

* **Mostrar Tudo (forEach)**: Percorre a lista de produtos e os renderiza na tela dinamicamente.
* **Mapear (map)**: Aplica um desconto de 10% em todos os itens do cardápio de uma só vez.
* **Somar Tudo (reduce)**: Calcula o valor total de todos os itens e exibe o resultado.
* **Filtrar (filter)**: Exibe apenas as opções de hambúrgueres veganos.

## 🎨 Design

A interface foi pensada para ser limpa e intuitiva, utilizando a paleta de cores vermelha e amarela, garantindo que o usuário identifique o propósito do site instantaneamente. O layout é organizado em colunas padronizadas para facilitar a visualização dos produtos.
